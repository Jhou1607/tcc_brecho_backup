🚀 INSTRUÇÕES RÁPIDAS - BRECHO LOOPLOOK 🚀

===========================================
📋 PRÉ-REQUISITOS
===========================================
✅ Node.js 18+ (https://nodejs.org/)
✅ PHP 8.2+ (https://php.net/)
✅ Composer (https://getcomposer.org/)
✅ PostgreSQL 12+ (https://www.postgresql.org/)
✅ Git (https://git-scm.com/)

===========================================
⚡ INSTALAÇÃO RÁPIDA
===========================================

1️⃣ BACKEND (Laravel)
----------------------
cd Backend
composer install
cp .env.example .env
php artisan key:generate

# Configurar banco no .env:
DB_CONNECTION=pgsql
DB_HOST=127.0.0.1
DB_PORT=5432
DB_DATABASE=tcc_brecho
DB_USERNAME=postgres
DB_PASSWORD=sua_senha

# Executar migrations e seeders:
php artisan migrate
php artisan db:seed
php artisan storage:link

# Iniciar servidor:
php artisan serve

2️⃣ FRONTEND (Angular)
-----------------------
cd Frontend
npm install

# Iniciar servidor:
ng serve

===========================================
🐛 PROBLEMAS COMUNS
===========================================

❌ Erro de conexão com banco:
   - Verificar se PostgreSQL está rodando
   - Criar banco: createdb tcc_brecho

❌ Erro de dependências:
   - Frontend: npm cache clean --force && npm install
   - Backend: composer clear-cache && composer install

❌ Erro de permissões (Linux/Mac):
   - chmod -R 755 storage bootstrap/cache
